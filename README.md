# PearlSmile Dental Hospital — Full Stack

## ✅ Your MongoDB credentials are already inside backend/.env

## HOW TO RUN

### Terminal 1 — Backend
```
cd backend
npm install
npm run dev
```
You should see: 🚀 PearlSmile API → http://localhost:5000

### Seed Doctor Accounts (ONE TIME ONLY)
Open your browser and go to:
http://localhost:5000/api/auth/seed-doctors

### Terminal 2 — Frontend
```
cd frontend
npm install
npm start
```
Website opens at http://localhost:3000
